## Prerequisites

- Node - `^v16.0`

## Getting Started

### Installing :wrench:

Paso a paso de cómo poner en marcha la aplicación en desarrollo.

### Clone the repo

```bash
git clone https://github.com/MarLuz/ort-taller-frontend.git
```

```bash
cd React/todo-app
```

### Install dependencies

```bash
npm install
```

### Initialize the dev server

```bash
npm start
```
